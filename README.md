# image-p
An image manipulation program

![](/docs/screenshots/interface.png)

# Features
- open/save images in: png, jpg, jpeg, bmp
- contrast
- hue
- brightness
- saturation
- grayscale
- invert
- darker
- brighter
- saturate
- desaturate
- binarize
- posterize
- gamma correction
- threshold
- sepia
- blur
- gaussian blur
- edge detection
- pixelize
